示範目錄
=======


請使用 [`vim -u .vimrc`](./.vimrc)
開啟位於 ["./repo/project"](./repo/project)
下的任意文件路徑，
並嘗試以下命令：

 命令                            | 方法參數                                     | \[執行文件參數\]
:----:                           |:--------:                                    |:------------
`CmdByURun`<br />`CmdByUHostRun` | `format`<br />`syntax`<br />`run`<br />`dev` | `golbal`<br />`g:cmdbyu_globalShFilePath`<br />`path/to/run.sh`
